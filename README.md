SATA-Unsupported
======================

An open source kernel for Hackintoshes with obscure SATA AHCI controllers.

#### Installation
The prebuilt binaries are available on [releases](https://github.com/extremegrief1/Ryzen-SATA-Unsupported/releases) page.
